# Public_CDN

jsdelivr,cdn 服务

# 创建一个公共仓库

把资源 push 到仓库，发布 releases（填写版本号）

# 示例

https://cdn.jsdelivr.net/gh/你的gihub账号/项目名称@发布的版本号/路径/文件名

# eg

- https://cdn.jsdelivr.net/gh/lightzhu/public_cdn@0.1/image/看点.png
